## 📌 프로젝트 목표

### 어떤 문제를 해결하나요?

혼자 사시는 부모님·조부모님의 안부가 걱정되지만, 전화하기는 서로 부담스러운 자녀 세대의 딜레마를 해결합니다.

### 기존 대안의 한계

| 방법 | 문제점 |
|------|--------|
| 📷 CCTV · 웹캠 | 감시받는 느낌 — 어르신이 설치 거부 |
| ⌚ 웨어러블 | 착용·충전·관리 불편, 사용률 7.1% |
| 📞 전화 | 자녀는 콜 포비아, 어르신은 잔소리 걱정 |

**CareMate는 카메라도, 착용 장치도, 전화도 없이** — mmWave 센서로 어르신의 재실·호흡·심박을 감지하고, AI가 자녀에게 자연어 브리핑으로 전달하는 비접촉 돌봄 서비스입니다.

## ✨ 주요 기능

### 1. mmWave 레이더 감지

- 카메라·착용 기기 없이 레이더 신호만으로 감지
- **감지 항목:** 재실 여부 / 센서로부터 거리 / 호흡량 / 심박수
- 1인 독거 가구 특화 — 단일 인원 감지에 최적화
- Wi-Fi CSI 방식 대비 데이터 안정성 우수

### 2. Edge AI 처리 (프라이버시 보호)

```
[원시 센서 데이터]
       ↓ (로컬 기기에서만 처리)
[정상 / 주의 / 경고 상태값]
       ↓ (이 값만 서버로 전송)
[자녀 앱에 표시]
```

> 원시 데이터는 기기 내부에서 즉시 폐기 — 촬영·녹화 없음

### 3. AI 개인화 브리핑

Claude API가 상태값 + 가족 입력 정보(이름·관계·특이사항)를 결합하여 수치 대신 자연어로 전달합니다.

| 상태 | 브리핑 예시 |
|------|------------|
| 🟢 정상 | "어머니께서 오늘 오전도 건강하게 활동 중이세요. 안심하세요 😊" |
| 🟡 주의 | "할머니께서 오후 1시부터 활동이 감지되지 않고 있어요. 한 번 연락해 보시는 건 어떨까요?" |
| 🔴 경고 | "아버지께서 오랫동안 활동이 감지되지 않고 있어요. 직접 연락하시거나 확인해 보세요." |

### 4. 상태 카드 UI

- 가족 상태를 친구 목록처럼 한눈에 확인
- **화면 흐름:** 로그인 → 메인 홈 → 상세 화면 → 장치 관리 → 마이페이지

## 🛠 기술 스택

### 시스템 아키텍처

```
[mmWave 센서 + ESP32]
       ↓ 원시 데이터 (로컬 처리 후 폐기)
[엣지 디바이스] — 상태값 판단 (정상/주의/경고)
       ↓ 상태값만 전송
[Spring Boot 서버] — 상태값 수신·저장 / 가족 그룹 관리
       ↓
[Claude API] — 상태값 + 가족 맥락 → 자연어 브리핑
       ↓
[React Native 앱] — 상태 카드 UI / 푸시 알림
```
